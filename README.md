# FeiraScore : Plataforma Digital de Avaliação para Feira Tecnológica
O FeiraScore busca tornar a feira tecnológica mais eficiente e participativa, substituindo a votação manual por um sistema digital que agiliza a apuração, aumenta o engajamento e melhora a experiência dos participantes, faça de uma forma mais jovem

## 📝 Objetivo 

* 🚀 Facilitar o processo de avaliação da feira tecnológica
* 📊 Divulgar a pontuação obtida de cada projeto avaliado
* 🗳️ Incluir democraticamente alunos e visitantes na votação
* 💻 Substituir os métodos manuais por meios mais tecnológicos
